# Week 12 - Payment Integration & E-commerce

---

## Task 1: Stripe Payment Integration

Built a Flutter app integrating Stripe for online payments. Implemented payment sheet, handled card transactions, created payment intents via backend, and stored transaction details in Firestore. Also handled success/failure states and tested using Stripe test cards.

---

## Task 2: Shopping Cart & Checkout Flow

Developed a complete e-commerce cart system with item management and quantity control. Implemented subtotal, tax, shipping, and discount calculations. Created a multi-step checkout flow including cart, address, payment, and order confirmation.

---

## Task 3: Order Management System

Created an order tracking system with Firestore integration. Displayed order history with status updates (pending to delivered), built detailed order views, and added cancel functionality. Included confirmation handling and refresh support.

---

## Task 4: Product Reviews & Ratings

Implemented a review system with star ratings and user comments. Calculated average ratings, displayed review lists, and added features like helpful votes, sorting, and filtering by rating.
